
   Utilities
   ---------------
   blosxom_mods ....... Mods to Blosxom blogger plugin to all proper use of 
                         Kwiki markup.
   cdmenu ............. Menuing System used for CD autostart.  Used in Expedite,
                         SiteXchange, and some other products.
   crypt .............. MFC Encryption app.
   CryptSetup ......... Setup for crypt.
   Data2Obj ........... Development utility: converts metadata in a config 
                         format into valid .obj files for linking.  This is a
                         pre-windows resource facility. *obsolete*
   DebugWindow ........ Utility for a popup window for perl debugging.  A perl
                         module plus a MFC status window.  Usefull for web 
                         script debugging.
   des ................ DES encryption console app.
   dump ............... Binary dump console app.
   DupeHD ............. Copy Hard disk contents.  low level *deprecated*
   EbsCom ............. EBS File read COM component.
   EbsComTest ......... Test apps for EbsCom
   EstimatorDataDump .. Console app to dump contents of an Estimator catalog.
   FieldnetEmulator ... Console app to test Fieldnet interfaces.
   GenPassword ........ Console app to create passwords.
   Grease ............. Gres like tool for replacing text in files.
   kwiki_mods ......... Mods to Kwiki to extend the kwiki markup language.
   QueryTSV ........... Multipurpose command line text querying utility.
   rc4_reentrant ...... Re-entrant form of RC4 API for use in FieldNet exits.
   SrcPack ............ Source Code pack utility
   SrcUnpak ........... Source Code unpack utility
   VSSToSVN ........... Convert VSS repositories to Subversion.  with samples.
